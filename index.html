Time Tracker

A command line application for tracking time during projects. This tool provides an easy way to track time while working, by using simple commands, such as start and stop which automatically store time and date in a local database.

This tool is perfect for a software developer/engineer who does contract work and would like an easy to control and use command line based, time tracking tool.

Time Tracker will keep track of one task at a time, but allows for easy switching between tasks, automatically stopping the clock as you switch.

Advanced users can create new functionality by accessing the advanced commands. For example it is possible to modify old times if there were mistakes.

Features

  • Uses sqlite database to store data
  • Designed to be easily scripted/automated
  • Database backup/restore function
  • Light weight / no background processes
  • Advanced functions for power users
  • Easy task creation and task switching
  • Summary command shows all task data and totals times
  • Backup and restore functionality included

Operating Systems

  • Windows
  • Linux

Examples

c:\>timetracker

TimeTracker:

    Easy task time tracking on the command line

Basic Usage:

    Start new task and the clock:   timetracker start "task name"
    Start the clock:                timetracker start
    Stop the clock:                 timetracker stop
    List all tasks:                 timetracker list
    Switch tasks:                   timetracker switch "task name"
    Delete task:                    timetracker delete "task name"
    Current task session info:      timetracker doc


optional arguments:
  -h, --help            show this help message and exit
  -v, --version         show version information
  -c, --config_dir      show config directory
  -e, --examples        show usage examples

Sub-Commands:
  Type  -h/--help for help on each command

  {start,stop,list,switch,delete,doc,task,session,clock,backup}
    start               start the task
    stop                stop the task
    list                list the tasks
    switch              switch the task
    delete              delete the task
    doc                 create documentation
    task                manage tasks
    session             manage sessions
    clock               manage the clock
    backup              backup database
